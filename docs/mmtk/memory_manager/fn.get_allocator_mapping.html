pub fn get_allocator_mapping<VM: VMBinding>(
    mmtk: &MMTK<VM>, 
    semantics: AllocationSemantics
) -> AllocatorSelector

Return an AllocatorSelector for the given allocation semantic. This method is provided so that VM compilers may call it to help generate allocation fast-path.

Arguments:

  • mmtk: The reference to an MMTk instance.
  • semantics: The allocation semantic to query.
