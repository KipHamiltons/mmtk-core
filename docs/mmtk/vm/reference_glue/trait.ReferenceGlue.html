    reference: ObjectReference,
    tls: VMWorkerThread
) -> ObjectReference[src]

Expand description

Process a reference with the current semantics and return an updated reference (e.g. with a new address) if the reference is still alive, otherwise return a null object reference.

Arguments:

  • trace: A reference to a TraceLocal object for this reference.
  • reference: The address of the reference. This may or may not be the address of a heap object, depending on the VM.
  • tls: The GC thread that is processing this reference.
